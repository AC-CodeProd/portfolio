# Welcome to My Personal Portfolio API
This is the official API documentation for My Personal Portfolio, a backend service for managing and showcasing personal portfolio data.

## Overview
My Personal Portfolio API provides endpoints to manage various aspects of a personal portfolio, including:
- Personal Information
- Projects
- Skills
- Experiences
- Education
- Technologies
- User Authentication and Administration

## Getting Started
To get started with the API, you can explore the available endpoints below. Each endpoint includes details about the request parameters, response formats, and authentication requirements.

## Authentication
The API uses Bearer Token authentication for most endpoints. You can obtain a token by logging in through the `/auth/login` endpoint.

## Rate Limiting
To ensure fair usage, the API implements rate limiting. Please refer to the rate limit headers in the responses for more information.
